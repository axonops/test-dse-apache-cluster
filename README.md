## Overview
Example of configurations for Executable jar.
## Building
Run next commands:

1. `cd ${projectDir}`
2. `mvn clean package`
3. `cd target`
4. `java -jar executable-jar-1.0-jar-with-dependencies.jar`

the result would:

`Hello world, executable-jar-example!`
